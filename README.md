# Analog Clock

A small web-based **Analog Clock** project built with **HTML, CSS, and JavaScript**.  
This project shows both an **analog clock** and a **digital clock** in a stylish UI.

## Features

- Analog clock with:
  - Hour hand
  - Minute hand
  - Second hand
- Digital clock display
- AM/PM indicator
- Modern neumorphic design
- Real-time clock updates using JavaScript
